### On Server

```
$ npm install
```

create a .env file with these attributes

```
PORT= [NUMBER]
JWT_SECRET = [STRING]
MONGO_URI=[MONGODB ADDRESS]
EMAIL_USERNAME=[EMAIL ADDRESS, STRING]
EMAIL_PASSWORD=[EMAIL PASSWORD, STRING]
```

then 

```
$ npm run dev
```

### On Client

``` 
$ npm install
$ npm run serve
```
